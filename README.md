# Secure Password Generator

A modern and responsive password generator built with **HTML, CSS, and JavaScript**.

This project allows users to create strong and secure passwords with customizable options and real-time password strength analysis.


## Features

- Generate secure random passwords
- Customize password length (8-32 characters)
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Password strength indicator
- Copy password to clipboard
- Responsive design for mobile and desktop devices
- Modern glassmorphism user interface


## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Git & GitHub
